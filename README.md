# C4A Kit de Madurez Digital

Este proyecto contiene la landing page oficial del Kit de Madurez Digital desarrollado por C4A.

## ¿Qué es el Kit?

Un conjunto de herramientas prácticas y visuales que ayudan a pymes, startups y software factories a diagnosticar, evaluar y mejorar su postura en ciberseguridad y gobernanza tecnológica.

## Contenidos clave

- Evaluación de madurez en 5 dominios esenciales
- Gráficos visuales y radiales comparativos
- Planes de acción sugeridos
- Plantillas descargables (.xlsx)

## Despliegue

Este proyecto usa Next.js con exportación estática y se despliega automáticamente en Cloudflare Pages.
